## Recommended AI Image Tools
- **Midjourney**: Use prompts above with `--ar 16:9` (feature graphic), `--ar 9:16` (screenshots)
- **DALL-E 3**: Paste prompts directly, request PNG output
- **Adobe Firefly**: Use for cleaner food illustrations
- **Canva AI**: Good for marketing/social media images with text overlay
